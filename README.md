# Daikin Airbase for Homey
This is a Homey App for the Daikin Airbase, utilising the [lachee/daikin-airbase](https://github.com/Lachee/daikin-airbase) library.
For further information on supported devices, please refer to the library.

- [Homey App Store](https://homey.app/a/dev.lachee.daikinairbase/test)
- [Daikin Airbase library](https://github.com/Lachee/daikin-airbase)
