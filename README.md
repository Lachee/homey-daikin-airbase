# Daikin Airbase for Homey
Turn your Daikin Airbase into a seamless part of your Homey setup.
This app builds on the [lachee/node-daikin-airbase](https://github.com/lachee/node-daikin-airbase) library to intergrate your Airbase App functionality into Homey.

With this app, you can manage your ducted climate system from Homey, including temperature, operating mode, fan behavior, and zone control. Whether you want comfort by schedule or smart reactions to your home’s activity, Homey Flows make it easy to automate everything.

Pair it with other devices to build smarter routines, like:
- Closing blinds when cooling starts
- Adjusting climate when you arrive home
- Coordinating temperature, lighting, and evening scenes


## Requirements

- A Daikin Airbase controller on the same local network as Homey
- Network access to the Airbase unit (manual IP setup is supported)

## Supported Device

- Any Daikin system that uses the [Airbase App](https://play.google.com/store/apps/details?id=au.com.daikin.airbase&hl=en_AU)
